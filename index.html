Add a new app
Drop a new folder with its own index.html — it shows up at its own link.
This is your personal hub at the root of the site. Each app lives in its own folder (e.g. golf/) and gets its own link like yoursite.github.io/golf/. To add one, create a new folder with an index.html inside, then add a matching card above. Tip: give each app its own browser-storage key so their data never collides.
